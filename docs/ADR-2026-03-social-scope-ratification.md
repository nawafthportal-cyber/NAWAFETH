# ADR: Social Scope Ratification

تاريخ القرار: 2026-03-17

## القرار
المنصة في نطاقها الحالي تُعامل كـ `socialized marketplace` وليست `community comments platform`.

## المعتمد في Sprint 1
- `follow / like / save / spotlights / portfolio / chat / reviews` تبقى ضمن الـ Social Layer الحالية.
- `Social comments` العامة مؤجلة حاليًا.

## الحالة
- `Defer`
- `Blocked by product decision` إذا طُلب تنفيذ comments عامة لاحقًا.

## الأثر
- يمنع توسع المنتج خارج النطاق الحالي.
- يسمح بإغلاق فجوة الـ Social Layer scope-wise في هذه المرحلة.
