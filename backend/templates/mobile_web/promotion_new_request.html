بنر الصفحة الرئيسية

واجهة بصرية رئيسية بحجم كبير لعرض التصميم أو الفيديو مع رابط توجيه مباشر.

قبل رفع مرفقات البنر:

الأنواع المدعومة: صور أو فيديو MP4 فقط. المقاس المعتمد 1920x840، وأي صورة أو فيديو MP4 يتم تجهيزه تلقائيًا وفق هذا المقاس عند الرفع.

الملفات المدعومة: JPG, JPEG, PNG, MP4. يتم ضبط الصورة أو الفيديو تلقائيًا قبل اعتماده إذا لزم الأمر.
معاينة البانر لم يتم اختيار ملف بعد
اختر صورة أو فيديو للمعاينة
سيتم اعتماد المقاس النظامي تلقائيًا للصور، بينما يجب أن يكون الفيديو بصيغة MP4 صالحة للعرض.

الرعاية

إظهار اسم وشعار الجهة الراعية ورسالتها ضمن حملة أطول مدى وأكثر ثباتًا.

ارفع شعارًا بجودة عالية وخلفية مناسبة للعرض داخل التطبيق. يفضّل تسمية الملف باسم الجهة الراعية لتسهيل المراجعة.

الملفات المدعومة: JPG, JPEG, PNG, MP4
يتم تحديدها تلقائيًا بعد اختيار البداية وعدد الأشهر.
